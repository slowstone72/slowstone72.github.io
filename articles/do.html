DelOS: A small joke program for Discord

"DelOS" is a small joke program I wrote for Discord, an online messaging service. To use it, you could include "delete dis" in a message, and the DelOS bot would send a message and use Discord's message editing feature to imitate typing into a "del" command into Command Prompt on Windows. Shortly after this, it'd delete your message from the chat.
