EpicOS: A bot for Multiplayer Piano

EpicOS is a command-based bot for Multiplayer Piano, originally created in 2015.

It's published on GitHub into the public domain under the Unlicense.

  • A rewritten userscript version of EpicOS (v1), created in 2024, optimized for modern browsers
